Acessibilidade na WEB e HTML

acessibilidade Web refere-se a prática inclusiva de fazer websites que possam ser utilizados por todas as pessoas que tenham deficiência ou não.

Atributo Alt nas Imagens.

Um site para ser considerado acessível deve ter todo o seu conteúdo em texto.

Atributo title (link).

Todos os links presentes em textos ou em páginas da Web devem deixar bem claro suas funções e objetivos.

Se criar uma lista, inserir marcadores

  • estabelece uma melhor organização
  • divisões bem definidas
  • facil de localizar as informações

Mais recomendações:

  • Não utilize somente a cor para transmitir informação.
  • verifique se os videos têm legendas.
  • Nunca sublinhar textos que não possuam link.
  • executar testes de navegação com o CSS Desabilitado.
  • Evitar menus com submenus, caso seja necessário, que possa ser acessível utilizando o teclado.
  • Verifique se a ampliação em 200% da página não perde informação.
  • Se possível, disponibilize uma visualização do site em modo noturno.
